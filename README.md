# 또이또미 과일 가게

수박 게임(Suika Game) 스타일의 과일 합치기 물리 퍼즐 게임입니다.
같은 과일끼리 부딪히면 더 큰 과일로 진화하며, 최고 점수를 노려보세요!

## 게임 화면

| 모바일 | 태블릿 | 데스크톱 |
|--------|--------|----------|
| 375px | 768px | 1920px |

## 실행 방법

별도 설치 없이 브라우저에서 바로 실행할 수 있습니다.

```bash
python3 -m http.server 8000
```

브라우저에서 `http://localhost:8000` 접속

또는 `index.html` 파일을 브라우저로 직접 열어도 됩니다.

## 조작법

| 입력 | 동작 |
|------|------|
| 마우스 이동 | 과일 좌우 이동 |
| 마우스 클릭 | 과일 떨어뜨리기 |
| 터치 드래그 | 과일 좌우 이동 (모바일) |
| 터치 해제 | 과일 떨어뜨리기 (모바일) |
| 방향키 좌/우, A/D | 과일 좌우 이동 |
| Space, 방향키 아래 | 과일 떨어뜨리기 |

## 과일 진화 순서

> 포도 → 딸기 → 귤 → 바나나 → 복숭아 → 메론 → 수박

두 개의 같은 과일이 부딪히면 다음 단계 과일로 합쳐지고, 점수를 획득합니다.

## 게임 오버 조건

과일이 LIMIT 선 위에 3초 이상 머물면 게임 오버!

## 기술 스택

- **HTML / CSS / JavaScript** (빌드 도구 없음)
- **Matter.js** (2D 물리 엔진, CDN)
- **Web Audio API** (효과음)
- **반응형 디자인**: Mobile First + `clamp()` + 3단계 breakpoint (모바일/태블릿/데스크톱)
